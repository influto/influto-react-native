# Security Policy

## Supported Versions

The latest published release of this InfluTo SDK receives security updates.
Please upgrade to the latest version before reporting an issue.

## Reporting a Vulnerability

Please report security vulnerabilities privately to **security@influ.to**.
Do **not** open a public GitHub issue for a suspected vulnerability.

Include, where possible:

- a description of the issue and its potential impact,
- steps to reproduce (a minimal proof of concept if possible),
- the affected SDK version(s) and platform.

We aim to acknowledge reports within 3 business days and to provide a
remediation timeline within 10 business days. Please give us a reasonable
opportunity to investigate and ship a fix before any public disclosure
(coordinated disclosure). We're happy to credit reporters who would like to
be acknowledged.

Thank you for helping keep InfluTo and its users safe.
